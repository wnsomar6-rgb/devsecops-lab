# DevSecOps Lab — CI/CD Secure Cloud Pipeline

## 🚀 Objectif du projet

Ce projet implémente une chaîne DevSecOps complète autour d'une API Python :

- Développement d'une API REST (FastAPI)
- Tests automatisés (pytest)
- CI/CD avec GitHub Actions
- Analyse de sécurité (CodeQL - SAST)
- Scan de vulnérabilités conteneurs (Trivy)
- Conteneurisation Docker
- Infrastructure as Code (Terraform)
- Déploiement Kubernetes (manifests)

---

## 🧱 Architecture
app/ -> API FastAPI
tests/ -> Tests automatisés
.github/ -> CI/CD GitHub Actions
docker/ -> Dockerfile
infra/terraform -> Infrastructure as Code
k8s/ -> Déploiement Kubernetes

---

## ⚙️ Pipeline DevSecOps

Le pipeline CI/CD automatise :

1. Tests unitaires
2. Analyse SAST (CodeQL)
3. Build Docker image
4. Scan sécurité (Trivy)
